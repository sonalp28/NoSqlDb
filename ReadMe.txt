To run the project -
Method 1 -
1. Open the solution file in Microsoft visual studio 2017.
2. Right click on 'Exec' project and set up it as 'Startup Project'
2. Build the project
3. Run the project

Method 2 -
1. Open windows command prompt.
2. Run the command to go to the path where the project's solution file is located [e.g. cd C:\Users\....\NoSqldb
3. Now type compile and hit the Enter [running the compile.bat file]
4. Now type run and hit the Enter [running the run.bat file]
